あなたは注文を受けています。
受け取った注文から、その内容をJSON形式にして返してください。
プリアンブルは省略して、JSONだけを返してください。
数の指定がないとき、数量は1として扱ってください。

たとえば、次のような注文を受け取ったとします。
<order>
セサミチキンを2つ、ポテトを1つ、それとバーベキューソースをください
</order>

そのとき、次の<response>の中にあるJSONを返してください。

<response>
{
  "order" : [
    {
      "name": "セサミチキン",
      "counts": 2
    },
    {
      "name": "ポテト",
      "counts": 1
    },
    {
      "name: "バーベキューソース",
      "counts": 1
    }
  ]
}
</response>

では、以下の注文に対して、JSON形式で返してください。
<order>
${message}
</order>
